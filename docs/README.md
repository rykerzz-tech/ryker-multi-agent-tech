# docs/

Documentation for `ryker-multi-agent-tech`.

## Conventions

- **plans/** — Roadmaps and feature plans. Move to `archive/` when fully shipped.
- **specs/** — Technical specifications that describe the current system. Update alongside code changes.
- **archive/** — Plans that are 100% completed. Keep for historical reference.
- **USAGE.md** — User-facing documentation. Lives at root for easy access.
